1. การประกาศตัวแปร (Variables)

ตัวแปรใน Java ต้องระบุชนิดข้อมูลก่อนเสมอ ตามรูปแบบ type variableName = value;

ชนิดตัวอย่างคำอธิบาย
char'J'ตัวอักษรเดี่ยว
String"Hello"ข้อความ
int3จำนวนเต็ม
float1.66fทศนิยม (ต้องมี f)
double3.345ทศนิยมความละเอียดสูง
finalfinal int X = 15;ค่าคงที่ (constant)
Java
// รูปแบบ: type variableName = value; char _one = 'J'; String _str = "osepha"; int This_Integer = 3; float This_Float = 1.66f; // ต้องมี f ต่อท้าย double This_Double = 3.3456; final int Const = 15; // final = ค่าคงที่
💡
Tip: ใช้ final เมื่อต้องการให้ค่าตัวแปรไม่เปลี่ยนแปลง เช่น ค่า PI
🖨️

2. การแสดงผล (Print Output)

Java มีคำสั่งแสดงผล 3 แบบ ได้แก่ print, println และ printf

print พิมพ์แบบไม่ขึ้นบรรทัดใหม่
Java
System.out.print("Hello"); // ไม่ขึ้นบรรทัดใหม่
println พิมพ์พร้อมขึ้นบรรทัดใหม่
Java
System.out.println("Hello"); // ขึ้นบรรทัดใหม่อัตโนมัติ System.out.println(3 + 15 + 3.34); // ผลลัพธ์: 21.34
printf พิมพ์แบบจัดรูปแบบ
Java
System.out.printf("Float = %f", This_Float); // %d = int, %f = float/double, %s = String, %c = char
🔄

3. การแปลงชนิดข้อมูล (Type Casting)

Widening แปลงอัตโนมัติ (เล็ก → ใหญ่)
byte→ short→ char→ int→ long→ float→ double
Java
int my_Int = 10; double my_Double = my_Int; // อัตโนมัติ: int → double System.out.println(my_Double); // 10.0
Narrowing แปลงด้วยตนเอง (ใหญ่ → เล็ก)
Java
double myD = 9.21; long myL = (long) myD; // ต้อง cast เอง: double → long System.out.println(myL); // 9 (ตัดทศนิยมทิ้ง)
⚠️
Warning: Narrowing Casting อาจทำให้สูญเสียข้อมูล เช่น ทศนิยมจะถูกตัดทิ้ง
📝

4. การจัดการ String

Methodคำอธิบายตัวอย่าง
.length()นับความยาว"Hello".length() → 5
.toUpperCase()แปลงเป็นตัวพิมพ์ใหญ่"abc" → "ABC"
.toLowerCase()แปลงเป็นตัวพิมพ์เล็ก"ABC" → "abc"
.indexOf()หาตำแหน่งตัวอักษร"Hello".indexOf("l") → 2
Java
String alpha = "AbCDEFGHiJKLMNOP"; System.out.println(alpha.length()); // 16 System.out.println(alpha.toUpperCase()); // ABCDEFGHIJKLMNOP System.out.println(alpha.toLowerCase()); // abcdefghijklmnop System.out.println(alpha.indexOf("G")); // 6
String Concatenation (การต่อ String)
Java
String s = "12"; int n = 20; String result = s + n; // ผลลัพธ์: "1220" (String)
📌
เมื่อ String + ตัวเลข จะได้ผลลัพธ์เป็น String เสมอ
🔢

5. Math Class

Methodคำอธิบาย
Math.random()สุ่มเลข 0.0 - 0.999...
Math.max(a,b)หาค่ามากสุด
Math.min(a,b)หาค่าน้อยสุด
Math.sqrt(x)รากที่ 2
Math.abs(x)ค่าสัมบูรณ์
Java
// สุ่มเลข 0 ถึง 100 int randomNum = (int) (Math.random() * 101); System.out.printf("Random: %d\n", randomNum);
📊

6. Array (อาร์เรย์)

Array เป็นโครงสร้างข้อมูลที่เก็บค่าหลายค่าในตัวแปรเดียว โดย index เริ่มจาก 0

Java
// ประกาศ Array String[] cars = {"Volvo", "BMW", "Ford", "Mazda"}; System.out.println(cars[0]); // Volvo System.out.println(cars[2]); // Ford // วนลูป Array for (String car : cars) { System.out.println(car); }
⚙️

7. Methods (เมธอด)

void Method ไม่คืนค่า
Java
// นิยาม Method static void myMethod(String lang, int num) { System.out.println(lang + " x " + num); } // เรียกใช้ myMethod("JAVA", 69); // JAVA x 69
return Method คืนค่า
Java
static int add(int x, int y) { return x + y; } int result = add(54, 56); // 110
Overloading Method ชื่อเดียวกัน ต่างพารามิเตอร์
Java
static int plus(int a, int b) { return a + b; } static double plus(double a, double b) { return a + b; } plus(88, 75); // เรียก int version → 163 plus(64.3, 36.26); // เรียก double version → 100.56
💡
Overloading: Java เลือก method ที่เหมาะสมจากชนิดข้อมูลของ argument โดยอัตโนมัติ
⌨️

8. Scanner (รับค่าจากผู้ใช้)

ใช้ Scanner class จาก java.util เพื่อรับข้อมูลจาก keyboard

Methodรับค่าประเภท
nextLine()String (ทั้งบรรทัด)
nextInt()int
nextDouble()double
nextBoolean()boolean
Java
import java.util.Scanner; Scanner myS = new Scanner(System.in); System.out.println("Enter username:"); String userName = myS.nextLine(); System.out.println("Username: " + userName);
⚠️
ระวัง: ใช้ nextLine() หลัง nextInt() จะต้อง nextLine() เปล่าก่อน 1 ครั้ง เพื่อเคลียร์ newline
